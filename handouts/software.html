Eclipse

Eclipse will be our development environment for both Java and Python. Eclipse has a significant number of features over Dr. Java with a number of useful tools including a debugger and easy generation of unit tests. There will be some sort of learning curve, but after a few weeks you should become quite comfortable with Eclipse as a development environment.

Go to the eclipse.org site and download the Eclipse installer:

Download the installer and run it. Click on the top button for Eclipse IDE for Java Developers:


This will install Eclipse in the location you specified with the installer.

(Windows Users should use the option to have a desktop shortcut.)

Python

Developing Python using Eclipse requires installing PyDev (We will be running Python 2.7)

Python 2.7 installed on OS X systems by default. Windows users may need to first download Python using the Windows Installer link on the python.org website.

*** PyDev requires Java 8 installation.

PyDev is installed from Eclipse. Go to

Help -> Install New Software

Enter http://pydev.org/updates in the Work with textbox. Next, click Add.


A dialog box will appear where you can name this site.  Give this the name PyDev.

Click on the button for Pydev

Python is a distinctly different programming language than Java. Although Python can be written using object-oriented techniques such as classes, the main distinction is that a given programming problem can be solved in significantly fewer lines of Python code. Python is not typed and allows variables to be declared dynamically. There are no begin-end squiggly braces, and indentation is used to identify blocks of code. By default, Python is loaded with several different data structures including lists, tuples, sets, and dictionaries. A good rule of thumb when developing in Python is to look for simple solutions to common programming problems; quite often we make things more difficult than necessary and Python allows a simple and elegant solution that may not be obvious at first to the beginning Python developer.

A great place to begin working with Python is the Python tutorial.
